# Морской бой (проект ВШЭ)

Проект по дисциплине **«Алгоритмизация и программирование»**

## Структура проекта
- **`utils.py`** — вспомогательные функции для работы с доской (создание, проверка, загрузка, сохранение).
- **`game.py`** — основная логика игры (класс `Game`, ходы игрока и компьютера).
- **`setup.py`** — модуль для ручной и автоматической расстановки кораблей.
- **`test_battleship.py`** — модульные тесты (запуск: `python3 -m pytest test_battleship.py`).
- **`requirements.txt`** — список зависимостей (пустой).

## Как запустить игру
1.  **Расставьте корабли:** запустите `python3 setup.py` и выберите вариант расстановки (автоматическая или ручная).
2.  **Запустите игру:** выполните `python3 game.py` и введите имя файла с сохранённой расстановкой.

## Документация
Исходный код полностью документирован в формате **reST** в соответствии с руководством кафедры. Документация в виде docstrings расположена в файлах `utils.py`, `game.py` и `setup.py`.

## Тестирование
Проект содержит модульные тесты для ключевых функций. Для их запуска выполните команду `python3 -m pytest` в корневой папке проекта.
